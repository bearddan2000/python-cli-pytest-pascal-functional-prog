# python-cli-pytest-pascal-functional-prog

## Description
Find the binomial coefficent.
Follows functional programming practices
POC for pytest.

## Tech stack
- python 3.8
  - pytest

## Docker stack
- python:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
